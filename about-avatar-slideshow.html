Custom List - Paste in a list of ENS names to create the avatar slideshow from. You should probably use the "Review Avatars" tab to review them first before starting the slideshow.
Custom List (prefetch all) - The same as "Custom List" except this prefetches the entire avatar image set once the slideshow begins. This is probably better for spotty network connectivity.
Review Avatars - Paste in a list of ENS names to review their avatars. Check if the avatar record is not set, the image broken, or too degen for a publicly shown slideshow.
CTRL-ALT-H - Use this hotkey to hide the top navbar. This way it will be invisible during the display of the slideshow of avatars. The hotkey can be used to unhide the top bar as well.
F11 - On most systems, you can press F11 on the keyboard to display the browser window in fullscreen mode so that there is no menu, address bar, etc. The slideshow should then take up the entire screen (if you've used CTRL-ALT-H to also hide the navbar)
default.png - This is the image that will display if the avatar is not set or is broken. Customize the image so it becomes the default when there is nothing to display. eg. image with company logo or event title, QR code to website, conference speaker schedule, etc. Most likely this image will not be in the rotatation because you should review the avatars for each ENS name. Do this in the "Review Avatars" tab.
background.jpg - This is the image that is used for the background while the slideshow of avatars is running.
